# Yaa-Huu Saloon - Online Booking Platform

Modern luxury salon booking platform built with React, TypeScript, and Vite. Features online appointment booking, service catalog, gallery, and reviews.

**Live Website:** https://yaahuusaloon.vercel.app/

## Features

- 📅 **Online Booking System** - Real-time appointment scheduling
- 💇 **Service Showcase** - Display all salon services
- 📸 **Lookbook Gallery** - Showcase styles and work
- ⭐ **Client Reviews** - Display testimonials
- 📱 **Mobile Responsive** - Works on all devices
- 🎨 **Luxury Design** - Premium aesthetic

## Run Locally

**Prerequisites:** Node.js (v16+)

1. Install dependencies:
   ```
   npm install
   ```
2. Set up environment variables in `.env.local`
3. Run the app:
   ```
   npm run dev
   ```
4. Open http://localhost:3000 in your browser

## Build for Production

```
npm run build
```
